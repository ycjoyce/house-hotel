# 好室House Hotel 旅館預約網站
好室House Hotel是一個基於 Vue.js 建構而成的訂房網站。UI 皆以 SCSS 遵循 OOCSS 手刻而成。

## 功能地圖
1. 首頁
- 可以看到所有房型
- 可以點擊任一房型，進入單一房型頁面查看更多介紹

2. 單一房型頁面
- 可以查看該房型詳細資訊，例如旅館描述、價格、時間、服務及房型照片
- 可以用日曆瀏覽該房型預約狀況並選擇預約
- 可以在選擇預約日期時，即時顯示訂房平假日天數及價格總價
- 可以填寫基本資訊訂房
- 可以回傳預約成功與否訊息

## 使用技術
- Vue.js / Vuex / Vue Router
- Webpack
- Javascript (ES6+)
- Ajax / Axios
- OOCSS
- SCSS
- ESLint (Airbnb)
